# Инструкция по тестированию обновлений профилей

## Запуск приложения

### Сервер
```bash
cd server
npm run start:dev
```
Сервер запустится на `http://localhost:3000`

### Клиент
```bash
cd client
npm run dev
```
Клиент запустится на `http://localhost:5173`

## Тест 1: Обновление профиля пользователя

1. Зарегистрируйтесь как пользователь или войдите:
   - Email: `alex@user.com`
   - Password: `123456`

2. Перейдите в "Мой профиль"

3. Нажмите "Редактировать профиль"

4. Измените имя и email

5. Нажмите "Сохранить"

**Ожидаемый результат:**
- ✅ Модальное окно закрывается
- ✅ Новые данные отображаются сразу БЕЗ перезагрузки страницы
- ✅ В консоли нет ошибок

## Тест 2: Обновление профиля тренера

1. Зарегистрируйтесь как тренер или войдите:
   - Email: `alexey.trainer@fitness.com`
   - Password: `trainer789`

2. Перейдите в "Мой профиль"

3. Нажмите "Редактировать профиль"

4. Измените:
   - Имя
   - Email
   - Описание
   - Загрузите новое фото профиля (опционально)

5. Нажмите "Сохранить"

**Ожидаемый результат:**
- ✅ Модальное окно закрывается
- ✅ Все изменения отображаются сразу БЕЗ перезагрузки
- ✅ Новое фото профиля видно сразу
- ✅ В консоли нет ошибок

## Тест 3: Добавление отзыва

1. Войдите как пользователь (не тренер)

2. Перейдите в "Тренеры" → Выберите любого тренера

3. Прокрутите до раздела "Отзывы"

4. Нажмите "Оставить отзыв"

5. Выберите оценку (звезды) и напишите комментарий

6. Нажмите "Отправить"

**Ожидаемый результат:**
- ✅ Форма закрывается
- ✅ Новый отзыв появляется в списке СРАЗУ
- ✅ Рейтинг тренера обновляется автоматически
- ✅ Страница НЕ перезагружается
- ✅ В консоли нет ошибок

## Проверка Redux DevTools

Если у вас установлен Redux DevTools:

1. Откройте DevTools → Redux

2. При обновлении профиля пользователя должен появиться action:
   - `user/update/fulfilled`

3. При обновлении профиля тренера:
   - `trainer/updateProfile/fulfilled`

4. При добавлении отзыва:
   - `trainerReview/addReview`

## Возможные проблемы

### Профиль не обновляется
- Проверьте консоль браузера на ошибки
- Убедитесь, что сервер запущен
- Проверьте, что токен авторизации валиден

### Отзыв не добавляется
- Убедитесь, что вы вошли как пользователь (не тренер)
- Проверьте, что `trainerReviewReducer` добавлен в store
- Проверьте консоль на ошибки API

### Изображения не загружаются
- Убедитесь, что папка `server/uploads` существует
- Проверьте права доступа к папке
- Проверьте CORS настройки сервера
