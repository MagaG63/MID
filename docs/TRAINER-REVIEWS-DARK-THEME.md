## Изменения

### Цвета

#### Фоны:
- **Форма отзыва**: `#f9f9f9` → `#1a1a1a`
- **Карточка отзыва**: `white` → `#202222`

- **Нет отзывов**: `#f9f9f9` → `#1a1a1a`

#### Границы:
- **Все границы**: `#ddd` / `#f0f0f0` → `#2d2d2d`

#### Текст:

- **Основной текст**: `#555` → `#a0a0a0`
- **Вторичный текст**: `#666` / `#999` → `#666` / `#a0a0a0`

#### Звезды:
- **Заполненные**: `#ffc107` (без изменений)
- **Пустые**: `#ddd` → `#2d2d2d`

#### Кнопки:
- **Primary градиент**: `#667eea → #764ba2` → `#20808d → #1a6b75`
- **Primary shadow**: `rgba(102, 126, 234, 0.3)` → `rgba(32, 128, 141, 0.4)`
- **Secondary**: `white` → `#252525`
- **Secondary border**: `#ddd` → `#2d2d2d`
- **Secondary hover**: `#667eea` → `#20808d`

#### Аватар:
- **Градиент**: `#667eea → #764ba2` → `#20808d → #1a6b75`

#### Сообщения:
- **Ошибка фон**: `#fee` → `#4d1a1a`
- **Ошибка граница**: `#fcc` → `#dc3545`
- **Ошибка текст**: `#c33` → `#ff6b6b`

#### Кнопка удаления:
