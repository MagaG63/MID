## Функциональность

### Для User
1. **Просмотр списка тренеров**
   - Видит всех тренеров
   - Видит рейтинг каждого
   - Видит программы тренировок
   - Может перейти в профиль

2. **Просмотр профиля тренера**
   - Видит полную информацию
   - Видит сертификаты
   - Видит все программы
   - Видит отзывы других пользователей
   - Может оставить свой отзыв

3. **Оставление отзыва**
   - Выбирает рейтинг (1-5 звезд)
   - Пишет текст (опционально)
   - Отправляет отзыв
   - Отзыв сразу появляется в списке

### Для Trainer
- Не имеет доступа к страницам тренеров
- Видит только свой профиль через `/profile`

### Для неавторизованных
- Не имеют доступа к страницам тренеров
- Перенаправляются на логин

## API Endpoints используемые

### Тренеры
- `GET /api/trainer` - список всех тренеров
- `GET /api/trainer/:id` - один тренер

### Отзывы
- `GET /api/trainer-reviews/trainer/:trainerId` - отзывы тренера
- `GET /api/trainer-reviews/trainer/:trainerId/rating` - рейтинг тренера
- `POST /api/trainer-reviews` - создать отзыв

### Программы
- `GET /api/training-program/trainer/:trainerId` - программы тренера

## Тестирование

### 1. Запусти сервер и клиент
```bash
# Сервер
cd server
npm run start

# Клиент
cd client
npm run dev
```

### 2. Зайди как user
1. Зарегистрируйся или войди как user
2. В навигации появится ссылка "Тренеры"
3. Нажми на "Тренеры"

### 3. Проверь список тренеров
- Видны карточки тренеров
- Показан рейтинг
- Показаны программы
- Работает кнопка "Посмотреть профиль"

### 4. Проверь профиль тренера
- Открывается профиль
- Видны фото и сертификаты
- Видны программы
- Видны отзывы
- Есть кнопка "Оставить отзыв"

### 5. Оставь отзыв
- Нажми "Оставить отзыв"
- Выбери рейтинг (звездочки)
- Напиши текст
- Нажми "Отправить"
- Отзыв появляется в списке

### 6. Проверь доступ
- Выйди из аккаунта
- Попробуй зайти на `/trainers` - должен перенаправить на логин
- Зайди как trainer
- Ссылки "Тренеры" не должно быть в навигации
